## ⚡ Features

*   **Fast & Lightweight:** Built using the latest version of Baileys for efficient resource management.
*   **Modular Command Handler:** Easily add, remove, or modify features without breaking the core system.
*   **Multi-Device Support:** Seamless connection via QR Code or Pairing Code.
*   **Media Handling:** Advanced processing for images, videos, audio, and documents.

## 🛠️ Installation & Setup

### Prerequisites

*   [Node.js](https://nodejs.org/) v18 or higher
*   [Git](https://git-scm.com/)
*   FFmpeg (for sticker & media conversion)

### Quick Start

1.  **Clone the Repository**
```bash
    git clone [https://github.com/your-username/kev-baileys.git](https://github.com/your-username/kev-baileys.git)
    cd kev-baileys
    ```

2.  **Install Dependencies**
```bash
    npm install
    ```

3.  **Configure Environment**
    Create a `.env` or `config.js` file to set your configuration (Owner number, prefix, etc.).

4.  **Run the Bot**
```bash
    npm start
    ```

## 💻 Deployment

This bot can be deployed on various cloud platforms or run locally:

*   **Local/VPS:** Run 24/7 using `pm2` (`pm2 start index.js --name "kev-bot"`).
*   **Termux:** Ideal for Android-based local hosting.
*   **Cloud Platforms:** Compatible with Railway, Koyeb, or Vercel (for API integration).

---

## ⚙️ Configuration

Inside your configuration file, you can customize:
*   `ownerNumber`: Your WhatsApp number.
*   `botName`: Default name for the bot (`KEV • BAILEYS`).
*   `prefix`: The symbol used to trigger commands (e.g., `.`, `!`, `/`).
