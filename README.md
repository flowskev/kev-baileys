## ⚡ Features

* **Fast & Lightweight:** Optimized for speed and low memory usage.
* **Modular System:** Plugin-based architecture for easy customization.
* **Multi-Device:** Stable connection using modern Baileys API.
* **Vercel Style:** Clean, minimalist documentation.
